# Mini Dashboard Projesi

Bu proje, Git ile ekip çalışması, push/pull döngüsü ve merge conflict yönetimini öğrenmek amacıyla hazırlanmıştır.

## Proje İçeriği

- weather.py → Hava durumu modülü
- news.py → Haber modülü
- main.py → Modülleri birleştiren ana dosya

## Merge Conflict Notları

- Yiğit (Öğrenci B): Push işlemi reddedildiğinde kısa bir karışıklık yaşadım ancak pull yapıp merge conflict'i manuel olarak çözdükten sonra Git’in iş birliği mantığını daha net anladım.

- Özlem (Öğrenci A): Güncel değişiklikleri pull ile çektiğimde kodların nasıl birleştiğini görmek ekip çalışmasının önemini anlamamı sağladı ve sürecin kontrollü ilerlediğini hissettim.
